## 2. 每个文件改动说明

### 2.1 `Shared/Models/AppSettings.swift`
- 新增 `monitoredDirectoryPaths: [String]` 配置字段。
- 默认值加入桌面路径（`~/Desktop`）。
- 增加兼容旧配置的解码逻辑：当旧 payload 不含该字段时自动回退默认桌面目录，避免升级后解码失败。

### 2.2 `App/ViewModels/MenuManagementViewModel.swift`
- 新增监听目录读写能力：
  - `monitoredDirectoryPaths`
  - `addMonitoredDirectory(path:)`
  - `removeMonitoredDirectories(atOffsets:)`
- 增加路径标准化与去重逻辑。
- 若删除后为空，自动回退桌面目录，避免 Extension 无监听目录。
- 补充日志：输出监听目录数量与路径，便于联调追踪。

### 2.3 `App/Settings/ApplicationSettingsView.swift`
- 新增“监听目录”配置区：
  - 显示当前目录列表
  - 支持删除目录
  - 支持“添加目录…”（文件夹选择器）
  - 支持快速添加 Desktop/Downloads/Documents
- 保持现有“全局设置/外部应用路径”结构不变，属于最小增量改动。

### 2.4 `RightClickFinderExtension/FinderSync.swift`
- 改为按共享配置动态设置 `FIFinderSyncController.default().directoryURLs`，不再硬编码仅桌面目录。
- 在 `init` 与 `menu(for:)` 时同步应用监听目录配置。
- 目录不存在/配置异常时自动回退到桌面目录。
- 补充监听目录日志输出（目录数量与路径）。

### 2.5 `Tests/UnitTests/UserDefaultsConfigurationStoreTests.swift`
- 新增 `testLoadLegacyPayloadWithoutMonitoredDirectories`，验证从旧配置升级时可正确回退到默认目录。
