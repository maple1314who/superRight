# 开发验证记录 V3.1.0

- 日期：2026-06-09
- 执行人：Codex
- 版本目标：把当前 UI 与核心右键功能从“界面可见”推进到“配置可保存、右键可执行、可回归验证”。

## 1. 本次完成范围

### 1.1 配置界面
- 重构主界面为侧边栏配置布局。
- 删除侧边栏中的“学习教程”“更多免费Apps”“联系我们”。
- 保留“订购”相关入口。
- 通用设置实现：
  - 隐藏 Dock 和菜单栏图标。
  - 完全磁盘权限状态刷新。
  - Finder 扩展启用说明。

### 1.2 IDEA 支持
- 新增 `ExternalApplication.idea`。
- 新增 `MenuActionType.openIdea`。
- 新增 `AppExecutionAction.openIdea`。
- Finder 右键菜单支持“用 IDEA 打开”。
- 配置界面支持 IDEA 路径配置。

### 1.3 新建文件
- 新增 `NewFileTemplateConfiguration`。
- 新建文件模板持久化到共享配置。
- Finder 右键菜单会根据启用模板动态生成新建文件动作。
- 新建文件动作由主 App 执行。
- 支持：
  - 默认文件名
  - 扩展名
  - 模板内容
  - 创建后打开
  - 创建后播放声音

### 1.4 发送文件到
- 新增 `FileDestinationConfiguration`。
- “发送文件到...”配置页接入真实共享配置。
- 支持新增、删除、重置目标目录。
- 支持编辑目标目录真实路径与显示名称。
- Finder 右键选中文件或文件夹时生成：
  - `复制到 <目录名>`
  - `移动到 <目录名>`（需开启）
- 复制/移动动作支持重名自动递增。
- Finder Extension 将复制/移动请求转发主 App 执行。

### 1.5 权限与运行
- 完全磁盘访问权限状态在 App 启动、激活、定时刷新时更新。
- 若权限缺失，App 会打开系统设置引导用户授权。
- App 图标资源已生成并接入资产目录。

## 4. 手工验证清单

### 4.1 新建文件
- 在监听目录空白处右键。
- 确认菜单出现已启用的新建文件模板。
- 点击 TXT/Markdown 等模板。
- 预期：文件创建成功，重名时自动生成 `Untitled 2.xxx`。

### 4.2 发送文件到
- 选中文件或文件夹右键。
- 确认出现“复制到 下载/图片/音乐/影片/文稿”。
- 点击其中一个目标。
- 预期：文件复制到目标目录，重名时自动递增。
- 打开“启用移动文件到...”后重复验证。
- 预期：出现“移动到 ...”，执行后源文件消失，目标目录出现文件。

### 4.3 IDEA
- 确认配置界面出现 IDEA 路径配置。
- 右键文件或文件夹。
- 点击“用 IDEA 打开”。
- 预期：IntelliJ IDEA 打开目标路径。

### 4.4 完全磁盘权限
- 系统设置中勾选完全磁盘访问权限。
- 重新激活 App。
- 预期：App 内权限状态刷新为已授权。

### 4.5 隐藏 Dock 和菜单栏图标
- 打开通用设置。
- 切换“隐藏 Dock 和菜单栏图标”。
- 预期：Dock 图标与状态栏图标按配置显示或隐藏。
