## 1. 本次完成范围

### 1.1 共享配置
- 新增 `ToolboxItemConfiguration`，保存工具箱功能项的启用状态、显示名称、排序、动作类型、选项、图标和颜色。
- `SharedConfiguration` 新增 `toolboxItems`，旧配置缺失该字段时自动补默认工具箱项。
- `AppSettings` 新增：
  - `showToolboxIcons`：控制配置界面是否显示工具箱图标。
  - `enableToolbox`：控制 Finder 右键菜单是否生成工具箱动作。
- `MenuActionType` 和 `AppExecutionAction` 新增工具箱基础动作：
  - `showFileInfo`
  - `copyFileName`
  - `createFolderFromFileName`
  - `hideSelectedItems`
  - `unhideSelectedItems`
  - `hideDirectoryItems`
  - `unhideDirectoryItems`

### 1.2 配置界面
- “工具箱”页面改为绑定真实共享配置。
- 支持启用/禁用工具箱项、编辑显示名称、重置默认工具箱项。
- 支持配置开关“显示图标”和“在 Finder 右键菜单启用工具箱”。

### 1.3 Finder 右键菜单
- 选中文件或文件夹时生成：
  - 文件信息
  - 拷贝文件(夹)名称
  - 根据文件名新建文件夹
  - 隐藏已选文件
  - 取消隐藏已选文件
- 在空白处或文件夹场景生成：
  - 隐藏全部文件
  - 取消隐藏全部文件
- Finder Extension 将工具箱动作转发给主 App，并携带当前目录、选中路径和工具箱选项。

### 1.4 主 App 执行
- `showFileInfo`：优先通过 Finder AppleScript 打开信息窗口；如果系统拒绝自动化权限，退化为 Finder 中定位选中项。
- `copyFileName`：将选中项名称写入剪贴板，一行一个名称。
- `createFolderFromFileName`：按选中项名称在其所在目录创建同名文件夹，冲突时自动追加序号。
- `hideSelectedItems` / `unhideSelectedItems`：设置选中项 Finder 隐藏标记。
- `hideDirectoryItems` / `unhideDirectoryItems`：设置当前目录直接子项 Finder 隐藏标记，不递归处理子目录内容。

## 4. 手工验证建议

1. 启动 `右键增强.app`，进入“工具箱”。
2. 确认工具箱默认项可见，启用状态、显示名称和重置可保存。
3. 在 Finder 选中文件右键，验证“拷贝文件(夹)名称”和“根据文件名新建文件夹”。
4. 在 Finder 选中文件右键，验证“隐藏已选文件”和“取消隐藏已选文件”。
5. 在目录空白处右键，验证“隐藏全部文件”和“取消隐藏全部文件”。
6. 验证“文件信息”；如果 macOS 弹出自动化权限提示，允许后再执行一次。

## 5. 已知限制

- 文件信息依赖 Finder AppleScript，首次使用可能触发自动化权限提示；被拒绝时只定位文件，不打开信息窗口。
- “隐藏全部文件”只处理当前目录直接子项，不递归隐藏子目录内容。
- 本次只实现高价值基础工具箱项，剪切、隔空投送、彻底删除、iShot、ICNS 转换仍未接入。
- Finder 右键菜单仍是扁平列表，尚未实现分组/子菜单语义。
