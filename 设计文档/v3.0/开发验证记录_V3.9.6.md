# 开发验证记录 V3.9.6

日期：2026-06-09

## 目标

补齐“新建文件”配置表中的“主菜单”开关语义，避免 UI 可勾选但 Finder 右键菜单无变化。

## 已完成

- `NewFileTemplateConfiguration.showInMainMenu` 通过 `MenuBuilder` 传递到 `MenuDisplayItem.isPromotedToMainMenu`。
- Finder Extension 构建根菜单时，会把勾选“主菜单”的动态模板项直接放到“右键增强”根菜单。
- 未勾选“主菜单”的模板项继续留在“新建”分类子菜单。
- 根菜单排序统一使用 `order`，因此主菜单模板和分类子菜单可以稳定混排。
- 主 App 与 Finder Extension 版本推进到 `3.9.6 / 2026060914`。

## 手动验证建议

1. 打开“新建文件”，勾选某个模板的“主菜单”。
2. 在 Finder 右键确认该模板直接显示在“右键增强”根菜单。
3. 取消勾选后再次右键，确认该模板回到“新建”子菜单。
4. 点击根菜单模板，确认仍按原模板内容创建文件。
