# 开发验证记录 V3.9.8

日期：2026-06-09

## 目标

修复配置界面标题栏仍占用顶部空间的问题，并调整“发送文件到”列表交互：行内容不可直接编辑，点击选择，按住拖拽排序。

## 已完成

- 主窗口启用 full-size content view，隐藏独立标题栏占位，让内容区延伸到窗口顶部。
- 保留系统红黄绿窗口按钮，并让按钮浮在左侧栏顶部区域。
- “发送文件到”列表行改为只读文本，避免误点进入编辑状态。
- “发送文件到”列表支持点击选择，删除按钮按当前选中项删除。
- “发送文件到”列表支持按住拖拽排序，并在 ViewModel 中统一重排 `order` 后持久化。
- 主 App 与 Finder Extension 版本推进到 `3.9.8 / 2026060916`。

## 手动验证建议

1. 打开主 App，确认顶部白色标题栏空间消失，红黄绿按钮位于左侧栏顶部。
2. 进入“发送文件到...”，点击任意行只选中，不进入路径或名称编辑。
3. 在“发送文件到...”中拖动行到其他位置，确认松手后排序稳定变化。
4. 重启 App 后确认“发送文件到...”排序持久化。
