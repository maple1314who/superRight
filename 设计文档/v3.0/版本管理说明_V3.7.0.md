# 版本管理说明 V3.7.0

- 日期：2026-06-09

## 1. 版本来源

主程序和 Finder Extension 的真实版本以 Xcode 工程构建配置为准：

- `MARKETING_VERSION`：用户可见版本号，对应 `CFBundleShortVersionString`。
- `CURRENT_PROJECT_VERSION`：构建号，对应 `CFBundleVersion`。

当前配置：

| Target | MARKETING_VERSION | CURRENT_PROJECT_VERSION |
| --- | --- | --- |
| 右键增强 | 3.7.0 | 2026060907 |
| RightClickFinderExtension | 3.7.0 | 2026060907 |

## 2. UI 展示规则

- 配置界面侧边栏固定展示应用名“右键增强”。
- 版本号从 `Bundle.main` 的 `CFBundleShortVersionString` 读取。
- 如果开发预览或测试环境无法读取 Bundle 版本，回退显示 `3.7.0`。
- 不允许在 UI 中再硬编码旧名称或旧版本号。

## 3. 发版规则

- 功能版本：递增 `MAJOR.MINOR.0`，例如 `3.7.0`。
- 修复版本：递增 `MAJOR.MINOR.PATCH`，例如 `3.7.1`。
- 构建号：使用日期加序号，格式为 `yyyyMMddNN`，例如 `2026060907`。
- 每次发版必须同步更新：
  - Xcode 工程 `MARKETING_VERSION`
  - Xcode 工程 `CURRENT_PROJECT_VERSION`
  - `CHANGELOG.md`
  - 对应开发验证记录
